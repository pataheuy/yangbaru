Halo! Saya pembuat Stikinot, seorang pelajar SMP yang suka ngulik komputer. Saya bikin aplikasi ini karena pengen punya catatan yang nggak ribet, bisa nempel terus di layar, dan punya timer buat belajar ala Pomodoro. Semuanya saya kerjain pakai HTML, CSS, dan JavaScript lewat framework Electron.

#HTML5 #TailwindCSS #JavaScript #ElectronJS

Fitur yang Bikin Kamu Makin Jago

Didesain simpel, berfungsi maksimal untuk kebutuhan harianmu.

📌

Pin Layar (Always on Top)

"Jangan biarkan catatanmu hilang tertutup jendela lain."

Gunakan fitur toggle Pin agar catatan tetap terlihat saat kamu ngetik di Word atau browsing.

✏️

Gaya Teks Lengkap

"Bikin catatan jadi lebih estetik dan gampang dibaca."

Mendukung Bold, Italic, Underline, hingga penomoran otomatis untuk tugas sekolah.

🔄

Auto-Save Instan

"Tiba-tiba mati lampu? Gak perlu panik lagi."

Setiap huruf yang kamu ketik otomatis tersimpan ke memori lokal tanpa perlu klik tombol simpan.

⏳

Pomodoro Spesial

"Belajar 25 menit, istirahat 5 menit. Fokus makin oke!"

Dilengkapi alarm (alarm.mp3) dan grafik visual biru yang keren buat nemenin kamu belajar.

🎨

Multi Warna Ceria

"Warna catatan sesuai dengan mood belajarmu hari ini."

Pilih 4 varian warna latar belakang yang soft dan nyaman di mata (Kuning, Pink, Hijau, Biru).

💻

Native Windows App

"Langsung jalan dari startup, nggak perlu buka browser."

Berjalan di System Tray (dekat jam) dan bisa otomatis terbuka saat komputer dinyalakan.

Teknologi yang Digunakan

Meskipun saya masih SMP, saya belajar kalau teknologi web itu luas banget. Saya menggabungkan kekuatan HTML untuk struktur dan CSS untuk tampilan yang bersih.

JavaScript

Otak utama aplikasi untuk hitungan timer dan logika simpan data otomatis.

Electron

Menyulap web buatan saya jadi aplikasi asli Windows (.exe) yang bisa nempel.

🚀
Ringan & Cepat

Hanya memakan sedikit RAM.
